Layout

1. Good Layout

I like this layout because it includes a lot of pictures, puts the most important things on top and then has slight variations throughout. The pictures make you excited to click on things in this layout.

2. Bad Layout

I don't like blackboard's layout because it has so much useless text. The only things I every do are click on the specific classes, which are tiny text links and then look at my grades. There's so much extra stuff that is unnecessary.
